# Demo Script

Target: **under 3 minutes**, clear before/after story, every claim traceable to a real source.

## 0. Hook (0:00–0:20)
- The problem in one line: 15–20% of cell lines are misidentified/contaminated; most researchers
  never check. State the stakes ($700M/yr, retracted papers, MDA-MB-435).

## 1. Quick Check (0:20–1:10)
- Search `MDA-MB-435` → red verdict.
- Show: true identity (melanoma, not breast), RRID, retracted papers, next steps.
- Emphasize source badges ("via Cellosaurus", "via ICLAC").

## 2. STR Test Reader (1:10–2:00)
- Paste an MCF-7 STR profile → green match, plain-language interpretation.
- Show a drift-vs-contamination explanation.

## 3. Methods Section Generator (2:00–2:40)
- Fill the form (MCF-7, ATCC HTB-22, Cancer Research) → publication-ready paragraph with RRID.
- Copy-to-clipboard, journal compliance check.

## 4. Close (2:40–3:00)
- Restate: weeks of confusion → minutes. Live deployed URL. Built with Claude.

## Test lines to demo
HeLa, MCF-7, MDA-MB-435, HEK293, MDA-MB-231 (see [tests/test_cell_lines.md](../tests/test_cell_lines.md)).
